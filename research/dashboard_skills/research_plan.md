# Dashboard skills research plan

## Main question
Which currently available Codex skills best support redesigning a Traditional Chinese judicial and public-opinion analytics dashboard, and how should each page use charts, tables, search, and AI summaries?

## Subtopics
1. Search current Codex curated and GitHub-hosted dashboard/UI skills; compare maintenance, adoption signals, scope, and install quality.
2. Review data-dashboard visualization practices for time series, categorical composition, searchable records, and public-opinion summaries.
3. Translate the findings into a four-page SPA information architecture and an implementation specification for this repository.

## Synthesis
Select one primary skill only when it adds capability beyond the already installed ui-ux-pro-max skill. Record sources and limitations. Use the resulting design system to implement and verify the local website.
